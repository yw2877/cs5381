## ✅ Your Task

Run the example scripts to practice making API calls before working with your own API.

### 🧱 Stage 1: Run curl Example

- [ ] Open **Git Bash** in the `01_query_api` folder
- [ ] Run: `bash 01_example_curl.sh`
- [ ] Review the JSON responses

### 🧱 Stage 2: Run Python or R Example

**Python:**
- [ ] Install: `pip install requests`
- [ ] Run: `python 02_example.py`

**R:**
- [ ] Install: `install.packages(c("httr2", "jsonlite"), repos = c(CRAN = "https://packagemanager.posit.co/cran/latest"))`
- [ ] Run: `source("02_example.R")` or `R -q -e "source('02_example.R')"`
